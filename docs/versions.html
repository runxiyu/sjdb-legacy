Change Logs

This page lists the process of the development of Daily Bulletin.

Alpha Test

  • Crucial 2023-09-13
    • Contents First demo finished manually, which contains most key sections.
    • Styling First demo finished manually, which contains the first CSS version.
  • Crucial 2023-09-18
    • Distribution First experiments with email distribution.
    • Programming Programs can now generate "On this Day", "In the News", and "Delicious Dinings".
    • Programming Programs can now turn HTML into EML for distribution.
    • Contents The exact date and a link to relevant Wikipedia pages are included in the section "On this Day".
    • Contents The section "Recommended by School" turns into "The School Recommends".
    • Styling Slight changes in maximum page width and font size.

Beta Test

  • Major 2023-09-19
    • Distribution Successful attempts at wider email distribution.
    • Programming A JSON file allow recipients to be automatically included in the EML.
    • Contents An introductory paragraph is included under the title.
    • Community A Microsoft Form is created to gather inspirations from the community.
    • Contents The day of the week under the title is displayed in a full word instead of an abbreviation.
    • Contents "Daily Inspiration" now features a flexible amount of inspirations.
  • Crucial 2023-09-21
    • Distribution First attempts at online versions are posted on https://users.andrewyu.org/.
  • Minor 2023-09-22
    • Styling Some paragraph breaks are replaced with <br> line breaks.
    • Styling The section "New Notices" is moved down.
  • Major 2023-09-25
    • Contents Part of "The School Recommends" is taken out as a new section "The School Inspires".
    • Styling Successful attempts at directly embedding pictures in HTML with base 64.
    • Contents Brief information about the past weekend is included in the section "On this Day".
    • Styling Paddings are added above headers for better visual experience.
    • Styling <time> element is utilized to formalize time representations.
  • Major 2023-09-26
    • Styling Google Fonts is used to enhance cross-platform experience.
    • Styling Padding above <h1> header is removed for better visual experience.
    • Styling Table columns now have a minimum width.
    • Styling "New Notices" is renamed as "Rather New Notices" and moved down when no longer new.
    • Styling Contents in <blockquote> are no longer all in italics.
  • Crucial 2023-10-09
    • Distribution Online versions are migrated to https://albertttan.github.io/daily-bulletin/.
    • Programming Programs can now automatically join the sections into an HTML file.
    • Contents Contents in Chinese are added: we are bilingual!
    • Styling Sans serif fonts are used throughout the page to improve uniformity and readability.
    • Community Pages that describe the project are created, including the one you are reading now.
    • Programming Programs can now generate "On this Day", "In the News", and "Delicious Dinings" in Chinese.
    • Programming Programs can now automatically update the tables.
    • Programming Programs can now automatically embed pictures in HTML with Base 64.
    • Contents A new section "Exam Reminders" will be included when a series of exams are going on.
    • Styling The CSS part is separated into a different file.
    • Styling A JavaScript allow table rows to collapse and expand when needed.
    • Community The Microsoft Form is split.
    • Contents The links to the online version are excluded from the online version.
    • Styling Parentheses marking locations are excluded from corresponding <span> elements.
  • Minor 2023-10-13
    • Contents The Chinese translation of "Quotes" is slightly altered.
  • Major 2023-10-16
    • Contents Recurring contents in "The School Inspires" and "The School Recommends" are removed.
    • Styling The margins of blockquotes are adjusted.
  • Minor 2023-10-23
    • Contents "Daily Inspiration" and "Promoted Contents" are swapped.
  • Minor 2023-10-25
    • Contents "Daily Inspiration" is moved down once more.

Regular Version

  • Major 2023-11-20
    • Distribution All students are added to the mailing list.
    • Contents "Promoted Contents" is removed.
    • Contents Snacks are added to "Delicious Dinings".
  • Major 2023-11-27
    • Contents "Delicious Dinings" is moved up.
  • Crucial 2023-11-29
    • Contents Deleted most inspirational contents inherited from The Week Ahead.
  • Minor 2024-03-18
    • Contents Inspiration sharers now have their Chinese names displayed as well.
